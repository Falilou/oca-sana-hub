# Dashboard Visualizations Directory

This directory stores Python-generated visualizations created from the Log Analysis Dashboard.

## Structure

```
dashboard-visualizations/
├── [timestamp_1]/
│   ├── Dashboard1_[timestamp_1].png
│   ├── Dashboard2_[timestamp_1].png
│   ├── analysis_summary.json
│   └── session_metadata.json
├── [timestamp_2]/
│   ├── Dashboard1_[timestamp_2].png
│   ├── Dashboard2_[timestamp_2].png
│   ├── analysis_summary.json
│   └── session_metadata.json
└── ...
```

## Files

- **Dashboard1_[timestamp].png**: Overview dashboard with 9 charts (top errors, timeline, hourly distribution, severity, category, operations, day of week, weekly, messages)
- **Dashboard2_[timestamp].png**: Detailed analytics dashboard with 6 charts (hourly heatmap, error types, box plot, peak vs off-peak, percentage breakdown, trend analysis)
- **analysis_summary.json**: Generated by Python script containing error statistics and file paths
- **session_metadata.json**: Generated by API endpoint containing request parameters and execution details

## Maintenance

Visualizations are stored with timestamps to prevent overwriting. Old visualizations can be manually deleted to save disk space.

## Access

All visualizations are publicly accessible via:
- `/dashboard-visualizations/[timestamp]/Dashboard1_[timestamp].png`
- `/dashboard-visualizations/[timestamp]/Dashboard2_[timestamp].png`
