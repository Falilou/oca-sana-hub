# 📱 OCA Sana Hub - Desktop Application

## User Guide for Colleagues

---

## 🎯 What Is This?

OCA Sana Hub is a desktop application that provides easy access to OCA Michelin e-ordering portals across 9 countries. Simply click on a country to access its portal!

---

## 💻 System Requirements

- **Windows:** Windows 10 or Windows 11
- **macOS:** macOS 10.13 (High Sierra) or later
- **Linux:** Most modern distributions (Ubuntu 18.04+, Fedora, etc.)
- **Internet:** Required to access portals
- **Hard Drive:** ~300 MB free space

**Note:** You do NOT need to install Node.js, npm, or any development tools!

---

## 🚀 How to Use

### First Time Setup

#### Windows (Installer Version)
1. Download: `OCA Sana Hub Setup 1.0.0.exe`
2. Double-click the installer
3. Follow installation wizard
4. Click "Finish"
5. Launch app from Start Menu or Desktop shortcut

#### Windows (Portable Version)
1. Download: `OCA Sana Hub 1.0.0.exe`
2. Save to a folder (e.g., Desktop or Documents)
3. Double-click to run
4. No installation needed!

#### macOS
1. Download: `OCA Sana Hub-1.0.0.dmg`
2. Double-click the DMG file
3. Drag "OCA Sana Hub" to Applications folder
4. Open from Applications

#### Linux
1. Download: `OCA Sana Hub-1.0.0.AppImage` or `.deb`
2. For AppImage:
   - Make executable: `chmod +x OCA*.AppImage`
   - Run: `./OCA*.AppImage`
3. For DEB:
   - Install: `sudo dpkg -i oca-sana-hub*.deb`

## 📖 Using the Application

### Main Dashboard

When you open the app, you'll see:

1. **Header Bar** (top)
   - Environment selector (PROD/INDUS)
   - Sana Sphere Tickets link
   - Theme toggle (light/dark mode)

2. **Country Portal Cards** (main area)
   - 9 countries available
   - Click any card to access that portal

3. **Sidebar** (left)
   - Dashboard (home)
   - Settings

### Accessing a Portal

1. **Select Environment:**
   - Use dropdown in top-right header
   - Choose "PROD" (production) or "INDUS" (testing)

2. **Click a Country:**
   - Click any country card
   - Modal window opens with portal details

3. **Access Portal:**
   - Click "Access Portal" button in modal
   - Portal opens in your default browser

### Available Countries

## ❓ Frequently Asked Questions

### General

**Q: Do I need internet to use this?**
A: Yes, internet is required to access the portals. The app itself works offline.

**Q: Can I use this on multiple computers?**
A: Yes! The portable version can be copied to any PC. Installers can be installed on any number of computers.

**Q: Do I need administrator rights?**
A: Installer version needs admin rights. Portable version does not.

### Technical

**Q: Is this a web browser?**
A: No, it's a portal hub. When you click "Access Portal," it opens in your default web browser.

**Q: Where are my portal credentials stored?**
A: The app doesn't store credentials. You log in directly on the portal website in your browser.

**Q: Can I customize portal URLs?**
A: URLs are configured during the app build. Contact your IT department for custom builds.

### Troubleshooting

**Q: The app won't start**
A: 
- Windows: Run as administrator
- Check antivirus isn't blocking it
- Verify you have Windows 10/11

**Q: Portal links don't work**
A:
- Check internet connection
- Verify portal URLs are accessible
- Try different environment (PROD/INDUS)

**Q: App is slow**
A:
- Close other programs
- Check internet speed
- Restart the application

**Q: Can't see country portal cards**
A:
- Scroll down on main dashboard
- Try resizing window
- Check zoom level (View menu)

## 🚨 Common Issues & Solutions

### Issue: White/Blank Screen
**Solution:**
1. Press Ctrl+R to reload
2. Close and reopen app
3. Check internet connection

### Issue: Portals Not Opening
**Solution:**
1. Verify default browser is set
2. Check firewall settings
3. Try different portal/environment

### Issue: Can't Switch Environments
**Solution:**
1. Click environment dropdown in header
2. Select desired environment
3. Portal cards will update

### Issue: Application Crashes
**Solution:**
1. Restart the application
2. Update to latest version
3. Contact IT support
